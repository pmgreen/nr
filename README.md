# nr
An interactive webmap of parcels managed or leased by the Narrow Ridge Center. Includes natural burial sites, trails, and roadways. Based on [this QGIS project](https://github.com/pmgreen/narrowridge) and was generated using the [qgis2web plugin](https://plugins.qgis.org/plugins/qgis2web/).
